Alumise taseme CASE-vahendid

Alumise taseme CASE-vahendid keskenduvad teostusele, kus mudelitest saab tegelik tarkvaratoode. Nad
toetavad andmebaasi struktuuri genereerimine, kood genereerimine, testide läbiviimist, koodi versioonhaldust
konfiguratsioonihaldust, pöördprojekteerimist.

Milliseid alumise taseme vahendeid olen kasutanud?

Kasutasin Visual Studio Code silmusi funktsiooni.

Millised vahendeid ei ole kasutanud? (kaks näidet)

1.Rational Software Architect

Rational Software Architect logo:

Rational Software Architect tööriista programmi akna pilt:

Rational Software Architect on terviklik disaini-, modelleerimis- ja arendustööriist tarkvara terviklikuks edastamiseks.
See kasutab ettevõtte Java rakenduste ja veebiteenuste kujundamiseks ühtset modelleerimiskeelt (UML).

2.Git

Git logo:

Git tööriista programmi akna pilt:

Git on hajutatud versioonikontrolli tarkvarasüsteem, mis on võimeline haldama lähtekoodi ja andmete versioone. Programmeerijad,
kes arendavad tarkvara koostöös, kasutavad seda sageli lähtekoodi haldamiseks.
